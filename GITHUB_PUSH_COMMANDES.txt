COMMANDES POUR ENVOYER SUR GITHUB

Ce dossier bot contient déjà un dépôt Git initialisé avec le remote:
https://github.com/znationcmd/extinction-plus-plus.git

Ouvre CMD ici:
cd C:\Users\utilisateur\Downloads\extinction-plus-plus-rss-github-ready\bot

Puis:
git status
git push -u origin main

Si GitHub refuse parce que le dépôt distant a déjà un historique différent:
git push -u origin main --force

Attention: --force remplace le contenu GitHub par ce dossier.

Railway:
Root Directory: bot
Build Command: npm run build
Start Command: npm start

Variables Railway:
DISCORD_TOKEN
CLIENT_ID
OWNER_ID
NITRADO_TOKEN
CLIENT_SECRET (si OAuth dashboard)
