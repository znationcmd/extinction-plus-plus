CORRECTION MOBILE

Cette version ajoute :
- menu mobile visible et cliquable
- barre rapide en bas
- page accueil avec boutons cliquables
- Whitelist accessible sur téléphone
- Shop/Serveurs/Killfeed accessibles sur téléphone

Après remplacement :
1. Ouvre le dossier dashboard
2. Lance :
   npm run build
   npm start
3. Sur téléphone :
   http://192.168.1.25:3000

Si le navigateur garde l'ancien affichage :
- ferme Safari complètement
- rouvre le lien
- ou teste directement :
  http://192.168.1.25:3000/whitelist
