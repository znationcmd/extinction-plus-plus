EXTINCTION++ RSS COMPLET - BOT + DASHBOARD + SHARED

Dossiers:
- bot        = Bot Discord
- dashboard  = Dashboard web
- shared     = Base de données commune

Installation rapide:
1. Ouvre bot/config.js
2. Mets DISCORD_TOKEN, CLIENT_ID, OWNER_ID
3. Double-clique start-bot.bat
4. Double-clique start-dashboard.bat
5. Dashboard: http://localhost:3000 ou 3001

Commandes importantes:
- /setup
- /serveur ajouter
- /serveur liste
- /shop create
- /shop list
- /shop buy
- /whitelist-panel
- /event kill
- /battlepass
- /quete liste

Important:
- Change les tokens/mots de passe déjà partagés.
- Le PC doit rester allumé si tu héberges localement.
