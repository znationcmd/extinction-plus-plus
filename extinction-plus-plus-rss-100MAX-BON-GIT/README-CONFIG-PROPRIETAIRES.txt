CONFIGURATION PROPRIÉTAIRES DISCORD

Ajouté :
- page /owner-config
- API /api/owner-config
- commande Discord /ownerconfig

But :
Chaque propriétaire Discord configure lui-même :
- son token Nitrado
- ses serveurs
- ses maps
- ses ID Nitrado
- IP / port / image
- activation shop / whitelist / battle pass

Toi, tu n’as pas besoin de remplir leurs infos dans Railway.

Railway garde seulement tes variables globales :
DISCORD_TOKEN
TOKEN
CLIENT_ID
DASHBOARD_URL
SHOP_URL
